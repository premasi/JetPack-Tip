## About Project <br />
Jetpack Tip App is a simple application to calculate tips and split bills.

## App Overview <br />
- Prerequisites
  - Android
  - Minimum Sdk 24 (Android 6.0)

- Installation
  - Download the APK
  - Install the APK
 
- How to Use
  - Open the app
  - Add the bill
  - Set the percentage to split bill
  - The app will calculate the tip automatically
